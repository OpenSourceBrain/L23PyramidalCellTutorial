How is this visualized on OSB?

The original BBP cell model (as well as their full neocortical microcircuit model) was developed to run in the NEURON simulator.

To allow the model to be used on Open Source Brain, it was converted to simulator independent NeuroML format (see technical details here), and that is what is being visualised in this interface.

To see the original XML code which generated this model click here.

However, one of the advantages of the model being in NeuroML and shared on OSB is that you don't need to look at the original source code to see the model structure and properties...
