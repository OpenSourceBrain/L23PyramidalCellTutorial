Simulating the electrical activity of the cell

The usual steps for running simulations are:

1) The user views the NeuroML model through their browser, potentially changing some parameter values

2) The model, changed parameter values and a description of what variables to record/how long to run the simulation for are sent to the OSB server

3) The OSB server will generate simulator code (usually for the NEURON simulator) and execute this, saving the results

4) The completed simulations show up in the Experiments tab at the bottom of the screen

Users need to be logged in to run their own simulations, but we have pre-recorded a few for you to look at...
